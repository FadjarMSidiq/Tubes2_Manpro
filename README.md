# Tubes2_Manpro
Lanjutan Tugas besar manpro
Repository ini digunakan sebagai pusat kolaborasi dalam pengembangan proyek. Seluruh anggota tim diharapkan mengikuti petunjuk dan struktur kerja berikut agar proses berjalan konsisten dan terorganisir.

Struktur Folder

#Struktur folder pada repository ini adalah sebagai berikut:

build/
Berisi hasil kompilasi sementara yang dihasilkan oleh sistem build. File dalam folder ini dibuat otomatis oleh IDE selama proses pengembangan.

dist/
Menyimpan hasil build final yang siap dijalankan atau didistribusikan. Biasanya berupa file executable atau paket aplikasi.

nbproject/
Berisi konfigurasi proyek bawaan NetBeans, termasuk pengaturan build, library, dan metadata proyek.

src/
Folder utama untuk seluruh source code aplikasi.

code program java/
Berisi seluruh file kode program Java yang menjadi inti pengembangan aplikasi.

test/
Folder untuk menyimpan kode pengujian (test cases) yang digunakan untuk memastikan fitur atau modul berjalan dengan benar.

#Alur Pengerjaan

1. Lakukan clone repository ini sebelum mulai bekerja.

2. Kerjakan perubahan pada branch terpisah untuk setiap fitur atau perbaikan.

3. Gunakan pesan commit yang jelas dan deskriptif.

4. Ajukan pull request setelah selesai agar dapat direview oleh PM.

#Catatan Tambahan

1. Jangan melakukan perubahan langsung pada branch utama tanpa persetujuan.

Jika mengalami kendala, buat issue baru agar dapat ditangani bersama.

Petunjuk tambahan akan disampaikan melalui file README atau melalui komunikasi tim.
